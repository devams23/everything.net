1.What is .NET?: 
	So .net is latest and open-source developer placeform created by Microsoft to build
	various Types of applications(web, mobile, desktop , gaming , cloud, etc) using different languages like C#, F#, VB.NET etc.

  
  
2.Difference between .NET Framework, .NET Core, and .NET (latest):
	.NET Framework: it was build primarily for Windows, and was closed source which supported desktop and web applications but was not cross-platform.
	
	.NET Core: cross-platform, open-source version of .NET that allows developers to build applications that run on Windows, Linux, and macOS. It is more modular and lightweight compared to ".NET Framework".
	
	.NET (latest): This is the unified and Open source platform/environment that combines the best features of .NET Framework and .NET Core. It supports cross-platform development and is designed to be the future of .NET development.
